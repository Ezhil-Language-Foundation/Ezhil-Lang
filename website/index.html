எழில் : தமிழ் நிரலாக்க மொழி
(Ezhil : Tamil programming language)

”எழில்” மொழியில் நிரல் எழுதிப் பழகுங்கள் (Try Ezhil Now!)

  1. எழில் என்பது தமிழில் கணினி நிரல்களை எழுத உதவும் ஒரு நிரலாக்க மொழி
    Ezhil is a Tamil programming language

  2. (English) ஆங்கிலம் அறியாதவர்கள் கணிப்பொறி இயக்கம் உதவும்
    English language is not a pre-requisite

  3. நிரலாக்கம் பற்றிக் கற்பதற்கு உதவும்
    N00b welcome

  4. எழில் உங்களுக்கு முதன்முறையாகத் நிரல்கள் எழுத உதவும்
    Write your first program in Ezhil

  5. இது ஒரு திறமூல (Open Source) மென்பொருள். (Free) இலவசமாக பயன்படுத்தலாம்
    Free as in Open, Free as in Beer

Ezhil is a Tamil programming language. Using Ezhil you can run a computer using instructions written in Tamil script. Ezhil includes both Tamil and English identifiers. Ezhil is intended to be useful for Tamil-speaking students who donot learn English early, but wish to learn programming for first time. Ezhil is released under GNU GPL license.

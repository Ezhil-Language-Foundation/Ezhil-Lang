Ezhil is a Tamil programming language. Using Ezhil you can run a computer using instructions written in Tamil script. Ezhil includes both Tamil and English identifiers. Ezhil is intended to be useful for Tamil-speaking students who donot learn English early, but wish to learn programming for first time. Ezhil is released under GNU GPL license.

Why Ezhil ?

Pre-processors நிரல் மொழி அல்ல. நமக்கு மொழியின் பொருளையும் அர்தத்தையும் கொண்ட ஒரு மொழி தேவை. இதற்கு தமிழ் நிரல் மொழி ஒன்று நிச்சயம் தேவை.

Pre-processors are not programming languages. We need to capture the semantics and the idea of computation through use of meta-syntactic variables. This requires a true programming language with an appropriate builtin functions, and API's.

எழில்: தமிழ் நிரலாக்க மொழி
(Ezhil : The Tamil programming language)

”எழில்” மொழியில் நிரல் எழுதிப் பழகுங்கள் (Try Ezhil Now!)

"எழில்" என்பது தமிழில் கணினி நிரல்களை எழுத உதவும் ஒரு நிரலாக்க மொழி. ஆங்கிலம் அறியாதவர்கள் கணிப்பொறி இயக்கம், நிரலாக்கம் பற்றிக் கற்பதற்கு உதவுவதும், அவர்களை முதன்முறையாகத் தமிழிலேயே நிரல்கள் எழுதச் செய்வதும்தான் இதன் நோக்கங்கள். இது ஒரு திறமூல (Open Source) மென்பொருள், இதனைப் பயன்படுத்துவோர் அனைவருக்கும் GNU GPL அனுமதியை வழங்குகிறது.

Ezhil is a Tamil programming language. Using Ezhil you can run a computer using instructions written in Tamil script. Ezhil includes both Tamil and English identifiers. Ezhil is intended to be useful for Tamil-speaking students who donot learn English early, but wish to learn programming for first time. Ezhil is released under GNU GPL license.

தமிழில் நிரல் மொழிகள் தேவையா? (Do you need Tamil Programming Language?)

Why Ezhil ?

கணினியில் நிரல் எழுத ஆங்கிலம் தேவை என்கிற நிலை இப்போது உள்ளது. ஆனால் உண்மையில், நிரல் எழுத ஆங்கிலத்தைவிட, தர்க்கரீதியிலான சிந்தனைதான் தேவை. ஆகவே, தமிழ்மட்டும் அறிந்தவர்களுக்கு இந்தச் சிந்தனையைச் சொல்லித்தர, அவர்கள் அறிந்த தமிழில் ஒரு நிரலாக்க மொழி நிச்சயம் தேவை.

Traditionally writing computer programming languages has required English language. However computers being symbolic machines, logical thought should be the only pre-requisite. Here comes Ezhil, to help the Tamil speaking programmer. Pre-processors are not programming languages. We need to capture the semantics and the idea of computation through use of meta-syntactic variables. This requires a true programming language with an appropriate builtin functions, and API's.
